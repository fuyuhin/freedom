###### freedom-微服务框架。

## Overview
- 集成iris
- 集成普罗米修斯
- 集成gorm
- 集成gcache
- 链路跟踪
- http2 server
- http2 client
- 依赖注入
- 组件总线
- CRUD 代码生成

## 安装
```sh
$ go get github.com/8treenet/freedom/freedom
```

## 创建项目
```sh
$ freedom new-project [project-name]
```
